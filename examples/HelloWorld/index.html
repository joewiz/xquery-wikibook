Discussion

        
The program creates a temporary variable called $message and assigns it a string value.
            The output is an XML element containing a message element which contains the value of
            the variable.

Plain Text

        
You can get XQuery to return plain text using serialization options which define the
            serialization and the output media-type.

        
For example to output the message as text, specify the serialization as text and the
            media-type as text/plain.
